# Skin Lesion Classification Using Smartphone Clinical Images and Patient Metadata

This project builds a skin lesion classification system using smartphone clinical images combined with patient metadata.  
The notebook covers data preparation, preprocessing, model training, and evaluation for lesion type prediction.

## Features
- Uses clinical image data for classification
- Incorporates patient metadata to improve performance
- End-to-end workflow in Jupyter Notebook

## Project Structure
- `skin-lesion-classification.ipynb` — main notebook with full pipeline

## Usage
1. Open the notebook in Jupyter/Colab.
2. Install required Python libraries.
3. Run cells sequentially to preprocess data, train the model, and evaluate results.

## Goal
To support early and accessible skin lesion screening using AI-based image classification.
